
# Репозиторий по проекту «Управление бронированием услуг гостиницы» с скриптами для баз данных, описанием проекта и файлами отчетов

Автор: Наталия Паламарчук

Добро пожаловать в репозиторий, посвящённый проекту «Управление бронированием услуг гостиницы»! Здесь вы найдёте скрипты для создания и наполнения баз данных в СУБД MS SQL Server, скрипты аналитических запросов, создания view и хранимых процедур, файлы отчетов, описание проекта.

## Структура репозитория
    
- **`diagram/`**  
  Папка с диаграммой базы данных.  

- **`docs/`**  
  Папка с описанием проекта.  

- **`reports/`**  
  Папка с файлами отчетов по проекту. Отчеты реализованы с помощью Power BI:  
  
  - **`Анализ доходности и эффективности.pbix`** — отчет для комплексного анализа работы отеля. Выручка за проживание и выручка за доп.услуги. Доход на один доступный номер. Лучший клиент отеля. Средняя цена за ночь. Сравнение плана и факта.  
  - **`Гео-аналитика.pbix`** — визуализация данных на карте: размер «пузырька» отеля — это его доход, а цвет — процент загрузки.  
  - **`Услуги и оплата.pbix`** — оплаты услуг по гостиницам в разрезе гостей, с фильтрами, мерами, диаграммами.  

- **`scripts/`**  
  Папка со скриптами для создания и наполнения базы данных. Также здесь можно найти скрипты для создания view и хранимой процедуры, скрипт аналитических запросов:  
  
  - **`create_db_and_tables.sql`** — скрипт для создания таблиц базы данных.  
  - **`create_triggers.sql`** — скрипт для создания двух триггеров: триггера проверки пересечения бронирований и триггера автоматической смены статуса номера на «Занят».  
  - **`insert_data.sql`** — скрипт для заполнения таблиц базы данных тестовыми данными.  
  - **`analytical queries.sql`** — скрипт, содержащий 6 сложных SQL-запросов для извлечения, агрегирования и анализа данных.  
  - **`create_views.sql`** — скрипт для создания 6 представлений, в том числе представления для отчета по гео-аналитике.  
  - **`сегментация данных.sql`** — скрипт, содержащий 3 SQL-запроса, реализующих задачу сегментации данных.  
  - **`create_stored_proc.sql`** — скрипт создания хранимой процедуры для отчета по анализу доходности и эффективности.  
  - **`исправление замечаний.sql`** — скрипт для исправления замечаний после проверки.
